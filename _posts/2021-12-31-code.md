List1 = [3, 4, 5, 20, 5]

* Find the index of the second 5
* Find the last element of this list
* Reverse this list



```python

List1 = [3,4,5,20,5]
index = [i for i, n in enumerate(List1) if n == 5][1]
print(index) 

# Find the last element of this list
print(List1[-1])

# Reverse this list
```
